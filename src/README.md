# Pasta src

Esta pasta foi criada para atender à estrutura mínima solicitada na avaliação.

Neste projeto, o código-fonte principal está organizado na pasta `app/`, seguindo a estrutura de um projeto Flask.

A aplicação utiliza:

- `app/` para o pacote principal da aplicação;
- `app/routes/` para as rotas e controllers;
- `templates/` para as páginas HTML;
- `static/` para arquivos CSS e estáticos;
- `config.py` para configurações;
- `run.py` para execução local.

Portanto, a pasta `src/` funciona apenas como referência estrutural, enquanto o código real permanece na organização original do projeto.
