# Evidências de Versionamento

Esta pasta reúne as evidências de uso de Git e GitHub no projeto Gestão RCC.

## Ciclo realizado

1. Caio organizou a estrutura inicial da avaliação.
2. Caio realizou commits e push para o GitHub.
3. Osvaldo baixou o repositório.
4. Osvaldo acessou a branch `docs/avaliacao-b1`.
5. Osvaldo realizou uma alteração própria.
6. Osvaldo fez commit e push.
7. Caio atualizou novamente o projeto com `git pull`.
